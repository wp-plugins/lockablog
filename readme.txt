This plugin allows you block content wordpress post for underage readers.

== Description ==

If you have content that you want protected from under age readers you can give the post a minimal age. On your website the readers have to give up there age
so they will only get to see the post if they are old enough.

== Installation ==
 
**Installation of Lock blog is to easy**
 
1. Upload the directory lockablog to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to settings then Twitme Settings and setup your account information and your done.
 
And your done :).
